Firebase Data Seeder

Seeds Firestore with Mumbai bus routes, stops & schedules. Run once per Firebase project.

⚠️

Before you start

Make sure Firebase is configured in firebase-config.js
  1. Edit firebase-config.js with your real Firebase credentials
  2. Enable Firestore and Realtime DB in your Firebase project
  3. Set Firestore rules to allow writes (dev mode)
  4. Click "Seed All Data" below
{
  "rules": {
    ".read": true,
    ".write": true
  }
}

⚠️ Switch to authenticated rules before going to production!
